I, Zombie reference executable

Run I-Zombie-Reference.exe in this folder.

Keep the lib folder and the assets folder next to the executable.
Do not copy the executable alone to another folder.
This package contains only the reference executable and runtime assets, not source code.

If Windows still reports a missing api-ms-win-crt*.dll file, install the
Microsoft Visual C++ Redistributable / Universal C Runtime for your system.
